# 💻 Mysite2026 - Web Development 

ยินดีต้อนรับสู่โปรเจกต์ **Mysite2026** หน้าเว็บแอปพลิเคชันรูปแบบ Modern Developer Portfolio/Landing Page ซึ่งเป็นส่วนหนึ่งของงานในรายวิชาการพัฒนาเว็บ (Web Development) 

---

## 🚀 ข้อมูลโปรเจกต์ (Project Overview)
* **ผู้พัฒนา:** `guccinet`
* **เทคโนโลยีที่ใช้:** Python, Django Web Framework, HTML5, CSS3
* **ฟีชเจอร์:** * หน้าจอ Landing Page สไตล์ Tech & Minimalist 
  * ระบบจัดการ Template และ Static Files ด้วย Django Architecture
  * หน้าจอ Responsive รองรับการแสดงผลทุกอุปกรณ์
